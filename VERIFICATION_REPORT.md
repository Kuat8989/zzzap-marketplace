## 🛠️ ЧТО НУЖНО СДЕЛАТЬ

### Шаг 1: Создать новые API ключи на Binance

1. Зайти на **Binance.com** → Account → API Management
2. Создать новый API ключ
3. **Обязательные разрешения**:
   - ✅ Enable Reading
   - ✅ Enable Spot Trading
   - ❌ Enable Withdrawals (НЕ включать для безопасности)
4. **IP Whitelist**: Либо добавить свой IP, либо оставить пустым (любой IP)

### Шаг 2: Обновить ключи в 2 местах

**Файл 1**: `~/.openclaw/workspace/HEARTBEAT.md` (строки 36-40)
```bash

```
