## Инструкция по созданию API ключей

**1. Зайди в настройки API:**
- Binance → Профиль (иконка) → **API Management**
- Или напрямую: `https://www.binance.com/ru/my/settings/api-management`

**2. Создай новый ключ:**
- Нажми **"Создать API"**
- Выбери **"System generated"** (сгенерированный системой)
- Придумай название (например: "OpenClaw Trading")
- Пройди верификацию (2FA)

**3. Настрой разрешения:**
- ✅ **Enable Reading** — обязательно
- ✅ **Enable Spot & Margin Trading** — для торговли
- ❌ **Enable Withdrawals** — НЕ включай! (безопасность)

**4. Ограничь по IP (опционально, но рекомендую):**
- Можешь указать IP для дополнительной безопасности

**5. Сохрани оба ключа:**
- **API Key** (публичный)
- **Secret Key** (приватный) — показывается только один раз!

---

⚠️ **Важно:** Secret Key покажется только при создании. Скопируй его сразу.
